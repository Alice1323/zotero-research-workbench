# User-confirmed document import modes

v0.3 will support three user-selected import modes for Document Candidates: Workbench-only storage, attaching a document to an existing Zotero item, and creating a new Zotero item with an attached document. This deliberately expands the project beyond the first-version read-only boundary, so every Zotero write must pass through a User Confirmation Gate that shows provenance, metadata, document count, and side effects before execution.
